Dynamic Host Configuration Protocol

## Ablauf

Client -------------------------> Server
						|
				DHCP-Discover (Broadcast)

Client <------------------------- Server
						|
					Offer

Client -------------------------> Server
						|
					Request

Client <------------------------- Server
						|
					ACK

* Ip-Adresse
* Netzmaske
* Default-Gateway
* DNS-Server
